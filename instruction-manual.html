2 Carton Type Options

Choose how sizes are packed inside the carton:

SINGLE SIZE

RATIO (MIXED)

  • Single: Enter just the size value (e.g., "XL").
  • Ratio: Enter quantity for each size (XS-3XL).

6 Printing Specs

  • Stock: Pressure sensitive label paper
  • Weight: 60 GSM or higher
  • Finish: Matte or low gloss
  • Opacity: 75% minimum

7 Troubleshooting

  • Text cut off: Add a space between words to allow wrapping.
  • Corner box missing: "H" box only for Homeware.
  • Wrong label size: Verify carton height.
  • Preview issues: Exported PDF is correct even if mobile preview is off.
  • Size too small: Ensure "Actual Size" print scale.
