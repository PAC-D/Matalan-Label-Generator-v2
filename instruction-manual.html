3 Carton Type Selection

Select the appropriate carton type based on the contents:

üì¶ Choose Carton Type

RATIO
Multiple Sizes

Carton contains mixed sizes
e.g., S-M-L-XL with quantities

SINGLE
One Size Only

Carton contains one size
e.g., just "L" or "XL"

Ratio: Enter quantity for each size (XS, S, M, L, XL, XXL, 3XL). The total will be calculated automatically.

Single: Enter just the size value (e.g., "M", "L", "XL", "42", etc.).

4 Important Notes

Carton gross weight (kg) will be measured after filling.

Printing Specifications

  • Barcodes and label should be printed on paper pressure sensitive label stock
  • Basis weight: 60 GSM or higher
  • Finish: Matte or low gloss
  • Opacity: 75%

5 Step-by-Step Guide

1

Open the Label Generator

Open the application in your web browser.

2

Select Label Size

Choose Custom for 10-15 cm cartons or A5 for all others.

3

Select Carton Type

Choose Ratio for mixed sizes or Single for one size only.

4

Enter All Required Fields

Fill in PO Number, Style Ref, Supplier, Factory, Line Code, Pack ID, Box Qty, Division, Color, Size, and Description.

5

Export PDF

Click "Export PDF" to download. Filename: [LineCode]_[PackID]_[Size].pdf

6 Troubleshooting

  • Text cut off: Add a space between the longest word. (This allows the text to wrap properly.)
  • Corner box not showing: This will only appear for Homeware division (H).
  • Wrong label size: Check the size selector and verify carton height.
  • Preview not updating: Refresh the browser page. On smaller screens, the preview might not be fully accurate‚Äîthe exported PDF will be accurate.
  • Size reduced after print: Print on A4 page. Set Scale to "Actual Size".
